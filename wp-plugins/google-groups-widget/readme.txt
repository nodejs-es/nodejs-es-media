The Google Groups widget simply pulls the latest discussions from your group into a nice display.

== Description ==

The Google Groups widget uses the built in wp_rss function to pull the top discussions from the Google Groups RSSv2.
It is then formatted from the css within it's ggroups directory to give it it's unique look. Mostly styled for non-IE
browsers.

== Installation ==

1. Upload `ggroups` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the "Google Groups Widget" into a sidebar, then edit the properties
